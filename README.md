SpecKit Plus – Short Note

SpecKit Plus aik tool hai jo software banane ke process ko step-by-step organize karta hai.
Is mein pehle idea ko samajh kar likha jata hai, phir us ka plan tayar hota hai.
Phir plan ko chhote tasks mein divide kiya jata hai aur akhir mein code likh kar project complete hota hai.
Yeh developer ko behtar planning aur behtar result hasil karne mein madad deta hai.

1️⃣ /constitution

/constitution wo step hai jahan project ke basic rules aur principles set kiye jate hain.
Is mein coding style, quality standards aur design rules decide hote hain.
Yeh project ke liye aik “qaida aur qanoon” ki tarah kaam karta hai.
Is se poora project aik hi direction follow karta hai aur consistency rehti hai.

2️⃣ /specify

/specify mein likha jata hai ke hum ne exact kya banana hai aur kyu.
Is stage mein user ki needs, features aur requirements simple zaban mein tayar ki jati hain.
Yeh step confusion door karta hai kyunke har cheez pehle se clear ho jati hai.
Isay aise samjhein: “kaam shuru se pehle hamein kya chahiye?”

3️⃣ /plan

/plan mein decide hota hai ke project kaise banega.
Is mein technology, design structure, data flow aur system architecture set kiye jate hain.
Yani specify stage mein jo chaha tha, plan stage us ka tareeqa banata hai.
Yeh step ensure karta hai ke development smooth aur organized ho.

4️⃣ /tasks

/tasks stage mein poore plan ko chhote chhote manageable steps mein tod diya jata hai.
Har task ka clear goal hota hai, taake developer ko pata ho pehle kya karna hai.
Yeh project ko easy, fast aur error-free banata hai.
Tasks ki wajah se workflow clear rehta hai aur time bhi bachta hai.

5️⃣ /implement

/implement woh step hai jahan asal coding shuru hoti hai.
Yahan developer tasks follow karte hue features banata hai.
Code test hota hai, improve hota hai aur akhir mein product tayar ho jata hai.
Yeh stage poore SpecKit Plus process ka final practical output hoti hai.
